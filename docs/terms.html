1. Service description

Valorant Stat Phone is a physical device (ESP32 + OLED display) paired with a web setup flow. After you voluntarily link your Riot account, the device may show statistics from your most recent VALORANT match (for example KDA). Data is fetched via authorized third-party APIs (Tracker Network and/or Riot Games API with RSO).
