Privacy Policy

Valorant Stat Phone respects your privacy. This product is not affiliated with Riot Games, Inc.

1. Data we collect

  • Device data: device ID, firmware version, pairing code, device secret, last seen timestamp.
  • Account data (opt-in): Riot ID, PUUID, OAuth tokens when using Riot Sign On.
  • Game statistics: recent match stats (KDA, map, agent).
  • Technical logs: basic server logs for security.

2. How we use data

To pair your device, fetch stats for the OLED, and operate the service. We do not sell your data.

3. Third parties

  • Riot Games — RSO and official APIs when enabled.
  • Tracker Network — player statistics when configured.
  • Hosting provider — backend deployment.

4. Retention and rights

Data is kept while your device is active. You may request deletion via GitHub issues or by resetting the device.

5. Security

Devices use unique secrets. Production uses HTTPS.
